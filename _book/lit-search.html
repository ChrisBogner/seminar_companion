3 Blended learning unit 2: Literature search

3.1 How to search for literature

3.1.1 Install a reference management software

Refer to chapter Software and install the reference management software. I recommend Zotero and can provide some support in case of major issues.

3.1.2 Data base Web of Science

Start by an overview of Web of Science (WoS): http://clarivate.libguides.com/webofscienceplatform. Then, create a profile. This allows you to save your searches and access from outside the UoC for 6 months. Watch the video below on how to create a profile.

Once you have registered and created a profile, start to search for literature. The two videos below show how to do this and provide tips on how to combine different key words for an effective search.

3.2 Assignments

  1. Create a new library (or collection in case you use Zotero) in your reference management software.

  2. Search WoS for at least 3 relevant publications on your seminar topic.

  3. Upload a screen shot of your bibliography in ILIAS.
